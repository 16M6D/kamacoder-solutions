## C++ 

```CPP 
#include<iostream>
using namespace std;
int main() {
    int a, b;
    while (cin >> a >> b) cout << a + b << endl;
}
```

## Java 

```Java
import java.lang.*;
import java.util.*;
 
public class Main{
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        while(in.hasNextInt()){
            int a = in.nextInt();
            int b = in.nextInt();
            System.out.println(a+b);
        }
    }
}
```

## python 

```python
import sys  
 
for line in sys.stdin:
    a, b = line.split(' ') 
    print(int(a) + int(b))
```
