
# 6. A+B问题VIII 

题目链接：https://kamacoder.com/problem.php?id=1005 

## C++ 

```CPP 
#include<iostream>
using namespace std;
int main() {
    int n, a, b;
    while (cin >> n) {
        while (n--) {
            cin >> a;
            int sum = 0;
            while (a--) {
                cin >> b;
                sum += b;
            }
            cout << sum << endl;
            if (n != 0) cout << endl;
        }
    }
}
```
