
# 9. 奇怪的信 

题目链接：https://kamacoder.com/problem.php?id=1008 

## C++ 

```CPP
#include<iostream>
using namespace std; 
int main() {
    int n, a;
    while (cin >> n) {
        int result = 0;
        while (n != 0) {
            a = (n % 10);
            n = n / 10;
            if (a % 2 == 0) result += a;
        }
        cout << result << endl;
        cout << endl;
    }
}
```
