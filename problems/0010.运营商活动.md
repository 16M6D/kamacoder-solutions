
# 10. 运营商活动

[题目链接](https://kamacoder.com/problem.php?id=1009)


## C++

```C++ 
#include<iostream>
using namespace std;
int main() {
    int m, k;
    while (cin >> m >> k) {
        if (m == 0 && k == 0) break;
        int sum = m + m / k; // 第一轮回得到总话费
        int count = m / k + m % k; // 额外话费 = 赠送+剩余话费 
        while (count / k != 0) { // 只要额外话费大于 k 就还能获得 剩余话费  
            sum += count / k;
            count = count / k + count % k;
        }
        cout << sum << endl;
    }
}


```
